نبذة عني

مرحبًا! أنا موسى عمر، مطور واجهات ويب محترف مع خبرة تزيد عن 5 سنوات في إنشاء حلول ويب مبتكرة تلبي احتياجات العملاء.

لقد عملت مع العديد من العملاء والشركات لتطوير حلول ويب مبتكرة تلبي احتياجاتهم.

المشاريع

بروفايل شركة تقنية

تصميم وتطوير موقع ويب تفاعلي لشركة تقنية ناشئة، يعرض خدماتها ومنتجاتها بطريقة مبتكرة.

التقنيات: React, Next.js, Tailwind CSS

منصة تعليم إلكتروني

تطوير منصة تعليمية تفاعلية تتيح للمستخدمين الوصول إلى الدورات وتتبع تقدمهم.

التقنيات: Vue.js, Vuex, Node.js, MongoDB

تطبيق إدارة المهام

تصميم وتطوير تطبيق ويب لإدارة المهام والمشاريع، مع ميزات التعاون وتتبع الوقت.

التقنيات: React, Redux, Firebase
